# US Flight Delay Analysis (2003–2022)  
![Dashboard 2003-2012](dashboard-(2003-2012).png)

## Overview
This project analyzes **U.S. flight delay data (2003–2022)** using interactive Excel dashboards.  
It focuses on **long-term trends, seasonal weather effects, airline performance**, and **geographic distribution of delays**.

Built as a **portfolio project**, it demonstrates **data modeling, DAX calculations, and analytical visualization** using **Excel Power Pivot**.

---

## Key Questions
- How have flight delays changed over time?
- Which seasons have the most weather-related delays?
- Which airlines have the lowest airline-caused delay rates?
- How do delays vary across U.S. locations?

## Key Takeaways
- Flight delays show strong long-term and seasonal patterns.
- Weather significantly impacts delays in specific seasons.
- Airline performance varies when measured by percentage-based delays.
- Delays are geographically concentrated rather than evenly distributed.
